# Video-game-data-analysis-project

## Overview

This project explores what factors influence global video game sales using Python for data analysis and Tableau for interactive visualization.

The analysis investigates relationships between critic scores, genre performance, regional preferences, and historical sales trends.

## Dataset

Source:
Video Games Sales (1980–2024)

Main fields used:
- Genre
- Critic Score
- Global Sales
- Regional Sales
- Release Date
